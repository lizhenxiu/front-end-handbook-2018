# 学习软件包管理器

> 软件包管理器或者是软件包管理系统是为计算机操作系统自动安装、升级、配置和卸载软件包的软件工具组合。它通常维护一个软件依赖关系和版本信息的数据库，用来防止软件不匹配和缺少依赖。

><cite>&#8212; [维基百科](https://en.wikipedia.org/wiki/Package_manager)</cite>
